     i. Sets `response.headers["X-Trace-ID"] = trace_id`
     j. Returns response

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP client -> FastAPI | Untrusted JSON request bodies cross into the server |
| FastAPI -> Supabase | Service role key sent to Supabase (trusted server-to-server) |
| FastAPI -> response | Server must not leak internal paths or tracebacks to client |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Spoofing | api/main.py | mitigate | CORS origin restriction via CORSMiddleware (allow_origins from Settings). No auth in Phase 4 MVP -- accepted for single-user. |
| T-04-02 | Tampering | api/snap.py | mitigate | Pydantic PanelsInput with strict=True, extra="forbid" validates all input at API boundary before pipeline code runs |
| T-04-03 | Information Disclosure | api/main.py | mitigate | Global exception handler returns ErrorResponse with error_type + message only. Full traceback logged server-side, never in HTTP response. |
| T-04-04 | Information Disclosure | .env / config.py | mitigate | .env gitignored. Service role key loaded via pydantic-settings, never exposed in responses or logs. .env.example has placeholder values only. |
| T-04-05 | Denial of Service | api/main.py | accept | Single-user MVP. Future mitigation: arq queue (D-14) when concurrent runs > 3 or run time > 60s. |
</threat_model>
