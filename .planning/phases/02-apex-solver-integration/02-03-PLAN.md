```python
def _point_to_segment_dist_xy(p, a, b) -> tuple[float, float]:
    p2, a2, b2 = p[:2], a[:2], b[:2]
    ab = b2 - a2
    denom = float(ab @ ab)
    if denom == 0.0:
        return float(np.linalg.norm(p2 - a2)), 0.0
    t = float((p2 - a2) @ ab) / denom
    t_clamped = max(0.0, min(1.0, t))
    closest = a2 + t_clamped * ab
    return float(np.linalg.norm(p2 - closest)), t_clamped
```

**Public function:**

```python
def densify_edges(
    polygons: dict[int, np.ndarray],
    planes: dict[int, Plane],
    graph: dict,
    tol: float = 1.0,
) -> dict[int, np.ndarray]:
    """Densify shared edges so adjacent panels carry the same vertex list.

    For each edge in the feature graph (panel_a, panel_b sharing 2+ features),
    collect all vertices from both panels that lie near the shared edge line,
    sort by parameter t, and insert missing vertices into each panel's polygon.
    Z is reconstructed per-panel via _z_on_plane.

    Copy-on-write: input polygons dict is never mutated.
    """
```

Algorithm (follows snapping.py densify_shared_edges_xy pattern):
1. `out = {pid: poly.copy() for pid, poly in polygons.items()}`
2. For each edge in `graph["edges"]`:
   - Get panel_a, panel_b from the edge.
   - Get feature_ids from the edge. Look up feature positions from graph["features"].
   - For each panel in (panel_a, panel_b):
     - Walk the polygon edges to find which edge(s) overlap with the other panel's vertices.
     - For each vertex of the OTHER panel that lies near this edge (XY distance < tol, 0.05 < t < 0.95 to avoid near-endpoints):
       - Skip if within tol of an existing vertex (endpoint dedup).
       - Project to the edge in XY, reconstruct Z via `_z_on_plane`.
       - Record (t, projected_xyz) as an insertion.
     - Sort insertions by t.
     - Insert into the polygon's vertex list.
   - Update `out[panel_a]` and `out[panel_b]`.
3. Log summary: `log.info("densified %d vertices across %d shared edges", insert_count, edge_count)`.
4. Return `out`.

Create `roof_pipeline/panel_snap_v2/validate.py` implementing `validate_polygons()`.

**Module structure** (follow winding.py pattern):

```python
"""Shapely polygon validation with graduated repair (TOPO-10, D-04, D-05, D-06).

Two-pass validation:
  Pass 1 (stage="solver"): Read-only diagnostic. is_valid + is_simple checks only.
    No repair. Failures log at DEBUG. This separates solver quality from densify quality.
  Pass 2 (stage="densify"): Correctness gate. make_valid() if needed. Area change
    tolerance: < 0.1% silent, 0.1-1% WARNING, >= 1% RuntimeError.
"""

from __future__ import annotations

import logging

import numpy as np
from shapely.geometry import Polygon, MultiPolygon
from shapely.validation import explain_validity, make_valid

from ..planes import Plane
from .winding import _project_to_2d

log = logging.getLogger(__name__)

# ---------------------------------------------------------------------------
# Constants
# ---------------------------------------------------------------------------

_AREA_WARN_THRESHOLD = 0.001    # 0.1%
_AREA_FAIL_THRESHOLD = 0.01    # 1.0%
_MULTI_AREA_RATIO_MIN = 0.95   # 95% -- below this, too much geometry lost
```

**Public function:**

```python
def validate_polygons(
    polygons: dict[int, np.ndarray],
    planes: dict[int, Plane],
    stage: str = "solver",
    repair: bool = False,
) -> dict[int, np.ndarray]:
    """Validate (and optionally repair) polygon geometry via Shapely.

    Args:
        polygons: Panel ID to (K, 3) vertices.
        planes: Panel ID to fitted Plane.
        stage: "solver" or "densify" -- used in log messages per D-04.
        repair: If True, attempt make_valid() on invalid polygons (pass 2 only).

    Returns:
        Copy of polygons, possibly with repaired geometry if repair=True.
    """
```

Algorithm:
1. `out = {pid: poly.copy() for pid, poly in polygons.items()}`
2. For each (pid, poly) in out.items():
   - Project to 2D: `verts_2d = _project_to_2d(poly, planes[pid])`
   - Build Shapely polygon: `shp = Polygon(verts_2d)`
   - If `shp.is_valid and shp.is_simple`: continue (clean polygon)
   - If NOT valid:
     - If `repair` is False (pass 1, stage="solver"):
       - `log.debug("panel %d invalid after %s: %s", pid, stage, explain_validity(shp))`
       - Continue -- do NOT modify the polygon
     - If `repair` is True (pass 2, stage="densify"):
       - `log.warning("panel %d invalid after %s, repairing: %s", pid, stage, explain_validity(shp))`
       - `repaired = make_valid(shp)`
       - **D-06 MultiPolygon handling:** If `isinstance(repaired, MultiPolygon)`:
         - `pieces = list(repaired.geoms)`
         - `largest = max(pieces, key=lambda g: g.area)`
         - `ratio = largest.area / repaired.area`
         - If `ratio < _MULTI_AREA_RATIO_MIN`:
           - Raise `RuntimeError(f"panel {pid}: make_valid produced MultiPolygon with largest piece ratio {ratio:.3f} < 0.95 -- too much geometry lost")`
         - Else: `log.warning("panel %d: MultiPolygon from make_valid, keeping largest piece (ratio=%.3f)", pid, ratio)`
         - `repaired = largest`
       - **D-05 Area change check:**
         - `original_area = abs(shp.area)` (use abs since invalid polygons can have negative area)
         - `repaired_area = repaired.area`
         - If `original_area > 0`: `delta_pct = abs(repaired_area - original_area) / original_area`
         - If `delta_pct >= _AREA_FAIL_THRESHOLD`:
           - Raise `RuntimeError(f"panel {pid}: repair changed polygon area by {delta_pct*100:.2f}% (old={original_area:.6f}, new={repaired_area:.6f}) -- likely cause: solver produced vertex outside original hull, or densify inserted vertex on wrong side of edge. Re-label panel boundary or reduce --snap-tol.")`
         - Elif `delta_pct >= _AREA_WARN_THRESHOLD`:
           - `log.warning("panel %d: repair changed area by %.3f%% (old=%.6f, new=%.6f)", pid, delta_pct*100, original_area, repaired_area)`
       - **Write back repaired coordinates:**
         - Extract coords from repaired polygon: `repaired_2d = np.array(repaired.exterior.coords[:-1])`
         - Reconstruct 3D from 2D using inverse projection (match nearest original vertex, same technique as winding.py). For each repaired 2D vertex, find the nearest original 2D vertex and use its 3D position. For vertices that were moved by make_valid (no close match), interpolate Z from the plane.
         - `out[pid] = repaired_3d`
3. Return `out`.

Note: The 3D reconstruction from repaired 2D coordinates should use a nearest-neighbor approach similar to winding.py's permutation recovery. Vertices that are close matches (within 1e-10 in 2D) use the original 3D position. Vertices moved by make_valid get their Z from `_z_on_plane`.
